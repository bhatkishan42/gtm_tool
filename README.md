# gtm_tool
-Automated Tag creation tool
+Automated GTM SOP builder for teams.

+## What it supports
+- Template-driven creation for tags, triggers, and variables.
+- Save your current setup as a reusable custom template.
+- Add templates directly in the app (stored in browser localStorage).
+- Build a multi-item SOP bundle and export **combined** JSON for all items.
+- One-click combined extract filename: `extract_all_tags_triggers_variables.json`.
